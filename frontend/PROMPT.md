# Ralph loop prompt — Corp LMS frontend

Изучи `@fix_plan.md` и `@DESIGN_SYSTEM.md`. Реализуй **следующий невыполненный** пункт `fix_plan.md`
(сверху вниз, по порядку секций 1→2→3→4).

Каждую итерацию:
1. Возьми первый `- [ ]` из `fix_plan.md`.
2. Реализуй его в коде (`src/`), переиспользуя компоненты `src/components/ui` и токены `globals.css`.
   Бренд KMG (teal #1896A7, Montserrat/Inter), русский текст, ничего не ведёт в Moodle, доступность.
3. Проверь, что фронт компилируется (dev-сервер) и нет ошибок; на ключевых экранах — нет горизонтального
   скролла на 375px.
4. Отметь пункт как `- [x]` в `fix_plan.md`.
5. Сделай git commit с понятным сообщением.
6. Если все пункты выполнены — выведи `EXIT_SIGNAL`.

Не меняй дизайн-токены и API компонентов без необходимости — только лейаут/экраны.
Не добавляй новый функционал бэкенда — только UI под существующие данные.
