# Fix Plan — Corp LMS frontend (учеников UI, бренд KMG)

Порядок: верхнее меню → футер → мобильный адаптив (приоритет) → контент-экраны.
Правила: бренд KMG (teal #1896A7, Montserrat/Inter), компоненты из `src/components/ui`,
русский текст, ничего не ведёт в Moodle, доступность (focus, контраст, тач ≥44px).

## 1. Каркас: верхнее меню + футер + мобильная навигация
- [x] Верхнее меню (Topbar/SiteHeader): десктоп — лого + профиль; мобайл — бургер + лого.
- [x] Мобильное меню: бургер → off-canvas drawer (`MobileNav`), затемнение, закрытие по тапу/Esc/смене маршрута, lock body-scroll.
- [x] Глобальный футер (`Footer`): бренд KMG, колонки (Платформа/Поддержка), языки KZ/RU/EN, копирайт.
- [x] Футер подключён в `(app)/layout.tsx`.

## 2. Мобильный адаптив (САМОЕ ВАЖНОЕ) — все экраны на 375px
- [x] Сайдбар: `hidden md:block`; на мобиле — drawer из бургера.
- [x] Дашборд/каталог/моё обучение: сетки 1→2→3→4→5 колонок.
- [x] Страница курса: грид `lg:grid-cols-[1fr_360px]` → на мобиле плеер сверху, программа снизу; вкладки скроллятся (`overflow-x-auto`).
- [x] Экран теста: карточки/кнопки на всю ширину.
- [x] Профиль/таблицы (Оценки): адаптивные — таблица на десктопе → карточки на мобиле.
- [ ] Финальная визуальная проверка на 375/768/1024/1440 (за пользователем).

- [x] Сертификаты: сетка `CertificateCard` + детальный экран `/certificates/[id]` + «Скачать PDF».
- [x] Календарь: ближайшие события (`core_calendar_*`) + empty state.
- [x] Оценки: таблица по курсам (`gradereport_user_get_grade_items`), адаптивная.

- [x] Задание: фокус-экран `/assignment/[id]/[cmid]` (описание, сдача текст/файл, статус); ссылка из программы курса в новой вкладке.
